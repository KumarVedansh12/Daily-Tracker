Features

📝 Daily Activity Logging (Study, Coding, Fitness, Sleep, Self Work, etc.)

📈 Interactive Charts

Daily bar chart

Enhanced line chart (Active hours vs Sleep)

Weekly stacked overview

Monthly averages dashboard

✅ Built-in To-Do List

Add, complete, and delete tasks

Tasks persist per day

🧠 Weekly Insights

Active hours

Consistency metrics

💾 Persistent Storage

Uses browser localStorage

No backend required

📤 CSV Export

Download all tracked data

📱 Responsive UI

Works on desktop and mobile

🌐 Deployed on GitHub Pages

🛠 Tech Stack

HTML5

CSS3 (custom dark UI)

JavaScript (Vanilla)

Chart.js for data visualization

GitHub Pages for deployment

🚀 Live Demo

🔗 https://kumarvedansh12.github.io/Daily-Tracker/

🎯 Purpose

This project was built to:

Practice real-world frontend architecture

Implement meaningful data visualization

Design a minimal yet functional UI

Create a deployable web application without frameworks
